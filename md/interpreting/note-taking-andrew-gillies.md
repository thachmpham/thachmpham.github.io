## Speech Analysis

:::::::::::::: {.columns}
::: {.column}

**Practice 1: Mind Map**

- Download transcripts.
- Mind map for the main ideas.

:::
::: {.column}

**Practice 2: Section Summary**

- Download transcripts.
- Two-column table.
    - Column 1: Transcript.
    - Column 2: One-line summary per section.

:::
::: {.column}

**Practice 3: Speech Summary**

- Listen to a speech.
    - How many points.
    - Note one word per point.
- Read transcript.
    - Highlight sentences sum up section.
