# Environment
## Linux
```sh
$ cat /etc/os-release
NAME="Fedora Linux"
VERSION="42 (Server Edition)"
RELEASE_TYPE=stable

$ pandoc --version
pandoc 3.1.11.1
Features: -server +lua
Scripting engine: Lua 5.4
```

```sh
$ apt install pandoc
$ apt install pandoc-plantuml-filter
```

## MacOS
```sh
$ brew install pandoc
$ brew install iproute2mac
```
