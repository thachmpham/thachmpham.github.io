1 Speech Analysis

1.1 Practice: Mind Map

  • Download transcripts.
  • Mind map for the main ideas.

1.2 Practice: Transcript Summary

  • Download transcripts.
  • Two-column table.
    • Transcript in column 1.
    • One-line summary per section in column 2.

1.3 Practice: Speech Summary

  • Listen to a speech.
    • How many points.
    • Note one word per point.
  • Read transcript.
    • Highlight sentences sum up the whole section.
